## Installation

 Just clone this repository and add two Environmental variables in your computer.

  Username | Password
  --- | ---
  *SECRET_USER* | `your_username`
  *SECRET_PASSWORD* | `your_password`
  *USER_EMAIL* | `email`
  *PASS_EMAIL* | `email_password`
